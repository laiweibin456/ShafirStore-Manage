### 4.3 升级逻辑详解

```java
/**
 * 完成预约订单 - 触发升级检查
 */
public Result<CompleteVO> completeOrder(Long orderId) {
    // 1. 更新订单状态为已完成
    ReservationOrder order = getById(orderId);
    order.setStatus(2);
    updateById(order);

    // 2. 获取用户信息
    Long userId = order.getUserId();
    Integer userType = order.getUserType();
    BigDecimal payAmount = order.getPayAmount();

    if (userType == 1) {
        // 普通用户：累加消费金额
        RegularUser user = regularUserService.getById(userId);
        BigDecimal newTotalConsume = user.getTotalConsume().add(payAmount);

        // 检查是否达到升级门槛
        MemberLevel nextLevel = memberLevelService.getNextLevel(500);

        if (nextLevel != null && newTotalConsume.compareTo(nextLevel.getMinAmount()) >= 0) {
            // 升级为会员
            upgradeToMember(user, nextLevel);
            return Result.success(CompleteVO.builder()
                    .upgraded(true)
                    .newLevel(nextLevel.getLevel())
                    .newLevelName(nextLevel.getName())
                    .build());
        } else {
            // 未达到升级，更新消费金额
            user.setTotalConsume(newTotalConsume);
            regularUserService.updateById(user);
        }
    } else {
        // 会员：累加消费金额，检查是否达到更高等级
        Member member = memberService.getById(userId);
        BigDecimal newTotalConsume = member.getTotalConsume().add(payAmount);

        // 更新会员消费和积分
        member.setTotalConsume(newTotalConsume);
        member.setPoints(member.getPoints() + payAmount.intValue());

        // 检查是否升级
        MemberLevel nextLevel = memberLevelService.getNextLevel(newTotalConsume);
        if (nextLevel != null && nextLevel.getLevel() > member.getLevel()) {
            member.setLevel(nextLevel.getLevel());
        }

        memberService.updateById(member);
    }

    return Result.success(CompleteVO.builder().upgraded(false).build());
}
```

### 7.2 后端 - ReservationOrderController.java

```java
package com.shafir.store.controller;

import com.shafir.store.common.result.Result;
import com.shafir.store.entity.ReservationOrder;
import com.shafir.store.service.ReservationOrderService;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.Map;

@Slf4j
@RestController
@RequestMapping("/api/reservation-order")
@RequiredArgsConstructor
public class ReservationOrderController {

    private final ReservationOrderService reservationOrderService;

    @PostMapping("/create")
    public Result<ReservationOrder> create(@RequestBody Map<String, Object> params) {
        List<Long> productIds = ((List<?>) params.get("productIds")).stream()
                .map(id -> Long.valueOf(id.toString())).toList();
        List<Integer> quantities = ((List<?>) params.get("quantities")).stream()
                .map(q -> Integer.valueOf(q.toString())).toList();
        String pickupTime = (String) params.get("pickupTime");
        String remark = (String) params.get("remark");

        ReservationOrder order = reservationOrderService.createOrder(
                productIds, quantities, pickupTime, remark);
        return Result.success(order);
    }

    @GetMapping("/list")
    public Result<List<ReservationOrder>> list(
            @RequestParam(required = false) Integer status) {
        List<ReservationOrder> orders = reservationOrderService.listByStatus(status);
        return Result.success(orders);
    }

    @GetMapping("/{id}")
    public Result<ReservationOrder> getById(@PathVariable Long id) {
        ReservationOrder order = reservationOrderService.getDetail(id);
        return Result.success(order);
    }

    @PutMapping("/{id}/complete")
    public Result<Map<String, Object>> complete(@PathVariable Long id) {
        Map<String, Object> result = reservationOrderService.completeOrder(id);
        return Result.success(result);
    }

    @PutMapping("/{id}/cancel")
    public Result<Boolean> cancel(@PathVariable Long id) {
        boolean result = reservationOrderService.cancelOrder(id);
        return Result.success(result);
    }
}
```

### 7.4 小程序 - 登录逻辑 (login.ts)

```typescript
import { miniLogin } from '../../../utils/request'

Page({
  data: {
    phone: '',
    password: ''
  },

  onPhoneInput(e: any) {
    this.setData({ phone: e.detail.value })
  },

  onPasswordInput(e: any) {
    this.setData({ password: e.detail.value })
  },

  async onLogin() {
    const { phone, password } = this.data

    if (!phone || phone.length !== 11) {
      wx.showToast({ title: '请输入正确手机号', icon: 'none' })
      return
    }

    if (!password) {
      wx.showToast({ title: '请输入密码', icon: 'none' })
      return
    }

    try {
      const res = await miniLogin(phone, password)

      wx.setStorageSync('token', res.data.token)
      wx.setStorageSync('userType', res.data.userType)
      wx.setStorageSync('userInfo', res.data.userInfo)

      wx.showToast({ title: '登录成功', icon: 'success' })

      setTimeout(() => {
        wx.switchTab({ url: '/pages/member/index/index' })
      }, 1500)
    } catch (err) {
      wx.showToast({ title: '登录失败', icon: 'none' })
    }
  },

  goToRegister() {
    wx.navigateTo({ url: '/pages/member/register/register' })
  }
})
```

## 十一、注意事项

### 11.1 安全事项

1. **密码加密**：后端存储密码建议使用 BCrypt 加密
2. **Token 验证**：所有需要登录的接口都需要验证 token
3. **防注入**：严格校验用户输入
4. **敏感信息**：手机号在展示时脱敏（138****8000）

### 11.2 性能优化

1. **数据库索引**：为常用查询字段添加索引
2. **分页查询**：订单列表必须分页
3. **缓存**：会员等级信息可缓存

### 11.3 异常处理

1. **重复注册**：手机号已存在时返回友好提示
2. **订单超时**：可设置预约订单超时自动取消
3. **库存不足**：创建订单时检查库存
