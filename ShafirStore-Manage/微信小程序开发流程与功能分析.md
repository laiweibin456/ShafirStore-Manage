## 二、阶段一：开发准备（第1-2周）

### 2.1 账号注册与资质申请

#### 2.1.1 微信小程序账号注册

| 步骤 | 操作 | 说明 | 时长 |
|------|------|------|------|
| 1 | 访问微信公众平台 | https://mp.weixin.qq.com | - |
| 2 | 选择"小程序"类型注册 | 使用未绑定公众号的邮箱 | 1天 |
| 3 | 主体信息登记 | 个人/企业主体选择 | 1天 |
| 4 | 完成认证 | 个人认证/企业认证 | 3-7天 |

#### 2.1.2 账号配置项

| 配置项 | 要求 | 说明 |
|-------|------|------|
| 小程序名称 | 莎菲尔菓子 | 需提前想好，不可重复 |
| 小程序图标 | 建议PNG 108x108 | 门店Logo |
| 服务类目 | 餐饮-烘焙糕点 | 根据实际选择 |
| 描述介绍 | 门店介绍文案 | 50-200字 |

#### 2.1.3 所需资质材料

| 主体类型 | 所需材料 | 费用 |
|---------|---------|------|
| 个人 | 身份证正反面 | 免费（个人主体功能受限） |
| 企业 | 营业执照、法人身份证 | ¥300/年认证费 |
| 个体工商户 | 营业执照 | ¥300/年认证费 |

### 2.2 开发环境搭建

#### 2.2.1 软件安装清单

| 软件 | 版本要求 | 用途 | 下载地址 |
|------|---------|------|----------|
| HBuilderX | 3.8+ | uni-app官方IDE | https://www.dcloud.io/hbuilderx.html |
| 微信开发者工具 | 1.09+ | 小程序调试预览 | https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html |
| Node.js | 18+ | npm包管理 | https://nodejs.org/ |
| VS Code | 最新版 | 辅助开发（可选） | https://code.visualstudio.com/ |

#### 2.2.2 环境配置步骤

```bash
# 1. 安装 Node.js 后验证
node -v
# 输出：v18.x.x

# 2. 配置 npm 镜像（国内加速）
npm config set registry https://registry.npmmirror.com

# 3. 全局安装 uni-app CLI
npm install -g @vue/cli @dcloudio/uni-cli

# 4. 验证 HBuilderX 可用性
# 打开 HBuilderX -> 工具 -> 插件安装 -> 安装 uni-app 插件
```

#### 2.2.3 项目初始化

```bash
# 方式一：使用 HBuilderX 可视化创建
# 文件 -> 新建 -> 项目 -> 选 uni-app -> Vue 3 版本

# 方式二：使用 CLI 命令行创建
npx degit dcloudio/uni-preset-vue#vite-ts shafirstore-mini

# 进入项目目录
cd shafirstore-mini

# 安装依赖
npm install

# 安装 uni-ui 组件库
npm install @dcloudio/uni-ui

# 安装 Pinia 状态管理
npm install pinia
```

### 2.3 微信公众平台配置

#### 2.3.1 服务器域名配置

登录微信公众平台 -> 开发管理 -> 开发设置，配置以下域名：

| 域名类型 | 配置项 | 说明 |
|---------|-------|------|
| request合法域名 | https://your-domain.com | 后端API地址 |
| uploadFile合法域名 | https://your-domain.com | 文件上传地址 |
| downloadFile合法域名 | https://your-domain.com | 文件下载地址 |

#### 2.3.2 AppID配置

在项目根目录创建 `manifest.json`，配置小程序的AppID：

```json
{
  "mp-weixin": {
    "appid": "wx1234567890abcdef",  // 替换为您的AppID
    "setting": {
      "urlCheck": false
    }
  }
}
```

#### 5.3.3 状态管理（store/cart.js）

```javascript
import { defineStore } from 'pinia'

export const useCartStore = defineStore('cart', {
  state: () => ({
    cartItems: [],  // 购物车商品列表
  }),

  getters: {
    // 选中的商品
    selectedItems: (state) => {
      return state.cartItems.filter(item => item.selected)
    },

    // 总数量
    totalCount: (state) => {
      return state.cartItems.reduce((sum, item) => sum + item.quantity, 0)
    },

    // 选中数量
    selectedCount: (state) => {
      return state.cartItems
        .filter(item => item.selected)
        .reduce((sum, item) => sum + item.quantity, 0)
    },

    // 总金额
    totalPrice: (state) => {
      return state.cartItems
        .filter(item => item.selected)
        .reduce((sum, item) => sum + item.price * item.quantity, 0)
    },

    // 是否全选
    isAllSelected: (state) => {
      return state.cartItems.length > 0 &&
        state.cartItems.every(item => item.selected)
    }
  },

  actions: {
    // 添加商品到购物车
    addItem(product, spec = null, quantity = 1) {
      const existIndex = this.cartItems.findIndex(
        item => item.productId === product.id && item.specId === spec?.id
      )

      if (existIndex > -1) {
        this.cartItems[existIndex].quantity += quantity
      } else {
        this.cartItems.push({
          id: Date.now(),
          productId: product.id,
          specId: spec?.id,
          name: product.name,
          price: product.price,
          image: product.image,
          specName: spec?.name,
          quantity,
          selected: true
        })
      }
    },

    // 更新数量
    updateQuantity(itemId, quantity) {
      const item = this.cartItems.find(item => item.id === itemId)
      if (item) {
        item.quantity = Math.max(1, quantity)
      }
    },

    // 删除商品
    removeItem(itemId) {
      const index = this.cartItems.findIndex(item => item.id === itemId)
      if (index > -1) {
        this.cartItems.splice(index, 1)
      }
    },

    // 选择/取消选择
    toggleSelect(itemId) {
      const item = this.cartItems.find(item => item.id === itemId)
      if (item) {
        item.selected = !item.selected
      }
    },

    // 全选/取消全选
    toggleAllSelected() {
      const allSelected = this.isAllSelected
      this.cartItems.forEach(item => {
        item.selected = !allSelected
      })
    },

    // 清空购物车
    clearCart() {
      this.cartItems = []
    }
  },

  // 持久化存储
  persist: {
    enabled: true,
    strategies: [
      {
        key: 'cart',
        storage: uni.getStorageSync ? uni.getStorageSync() : localStorage,
      }
    ]
  }
})
```

## 六、API接口对接

### 6.1 请求封装（api/request.js）

```javascript
// 封装请求方法，复用现有后端API理念
const BASE_URL = 'https://your-domain.com/api'

const request = (options) => {
  return new Promise((resolve, reject) => {
    // 获取Token
    const token = uni.getStorageSync('token')

    uni.request({
      url: BASE_URL + options.url,
      method: options.method || 'GET',
      data: options.data || {},
      header: {
        'Content-Type': 'application/json',
        'Authorization': token ? `Bearer ${token}` : '',
        ...options.header
      },
      success: (res) => {
        if (res.statusCode === 200) {
          // 匹配现有后端的统一响应格式
          if (res.data.code === 200) {
            resolve(res.data)
          } else if (res.data.code === 401) {
            // Token过期，跳转登录
            uni.removeStorageSync('token')
            uni.navigateTo({ url: '/pages/user/login/index' })
            reject(res.data)
          } else {
            uni.showToast({
              title: res.data.message || '请求失败',
              icon: 'none'
            })
            reject(res.data)
          }
        } else {
          uni.showToast({
            title: '网络请求失败',
            icon: 'none'
          })
          reject(res)
        }
      },
      fail: (err) => {
        uni.showToast({
          title: '网络错误',
          icon: 'none'
        })
        reject(err)
      }
    })
  })
}

export default request
```
